# Graffiti / WoW conjecture 698 — PROVED TRUE

Self-contained verification package for a math researcher.

## The claim (now a theorem)

From Fajtlowicz's *Written on the Wall* (`wow-july2004.pdf`, page 104, item 698;
the PDF has broken Type-3 fonts — wording verified glyph-by-glyph):

> 698. lenght [sic] of negative eigenvalues ≤ the Randic Index.

Per the WoW / Brewster–Dinneen–Faber glossary: *length* = Euclidean (L2) norm,
*eigenvalues* = adjacency eigenvalues with multiplicity. So the claim is

  s⁻(G) := √( Σ_{λᵢ<0} λᵢ² ) ≤ R(G) := Σ_{uv∈E} (dᵤ dᵥ)^(−1/2)

for every finite simple graph. **This is TRUE**, with equality iff the graph is a
complete bipartite graph plus isolated vertices (or empty). Proof: `PROOF-698.md` —
a four-step chain: λ₁ ≥ S/m (Rayleigh at x = (√dᵤ)), m² ≤ S·R (Cauchy–Schwarz over
edges), λ₁² + R² ≥ 2λ₁R ≥ 2m (AM–GM), and Σλᵢ² = trace(A²) = 2m giving
s⁻² ≤ 2m − λ₁² ≤ R².

## Provenance and status

- Conjecture generated by Fajtlowicz's Graffiti program (~1989/90 block of WoW);
  unannotated in the July 2004 WoW revision, i.e. open there.
- Roucairol–Cazenave (ECAI 2025; `roucairol-cazenave-2025.pdf` is their earlier
  arXiv version) list 698 as Open. Notably their Rust encoding of 698 used the
  **Laplacian** spectrum, which is PSD — no negative eigenvalues — so their
  machine search for this item was vacuous.
- The two lemmas powering the proof are Favaron–Mahéo–Saclé 1993 (Discrete
  Math. 111): λ₁ ≥ S/m and λ₁·R ≥ m. The assembly into s⁻ ≤ R and the equality
  characterization appear to be new; closest published relative is
  Arizmendi–Arizmendi (LAA 609 (2020)): E(G) ≥ 2R(G) (L1-norm analogue; neither
  implies the other). Full priority discussion and residual risks (notably: the
  paywalled FMS 1993 primary source was read only via surveys):
  `ADVERSARIAL-REVIEW.md` §3.

## How to verify

- `python3 verify.py` — exact-arithmetic checks of the equality families and the
  whole chain on all 2^15 labeled n=6 graphs + 4000 random graphs. Expect PASS.
- `python3 verify_adversarial.py` — the hostile reviewer's independent verifier
  (exact sympy characteristic polynomials, all 2^21 labeled n≤7 graphs, random
  n≤70, 50-digit recertification of near-equality cases). Expect ADVERSARIAL-PASS.
- Run notes (searches, dead ends, the WoW 129 frontier work): `RUN-NOTES.md`.
- Independent adversarial re-derivation + statement fidelity + priority check:
  `ADVERSARIAL-REVIEW.md`.

## Lean formalization

`lean/` — the full theorem is machine-checked in Lean 4 + mathlib:
`P06.wow_conjecture_698` (no connectivity/m ≥ 1 hypothesis; isolated vertices and
the empty graph included). No `sorry`, no `native_decide`, no added axioms:
`#print axioms P06.wow_conjecture_698` = `[propext, Classical.choice, Quot.sound]`.
The equality characterization is not formalized (not needed to settle 698).

Build: `cd lean && lake exe cache get && lake build`
(this directory mirrors `formalization/P06` in the main repo).
