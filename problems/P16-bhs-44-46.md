# P16 — BHS Laplacian bounds #44 and #46 (the last two open of 68)

**Statement.** For every connected graph G on ≥ 2 vertices, with dᵢ = degree and mᵢ = average
degree of the neighbors of vertex i, do the following hold, where μ(G) is the largest Laplacian
eigenvalue?

- **Bound 44:** μ(G) ≤ max_{i∼j} [ 2 + √( 2( (dᵢ−1)² + (dⱼ−1)² + mᵢmⱼ − dᵢdⱼ ) ) ]
- **Bound 46:** μ(G) ≤ max_{i∼j} [ 2 + √( 2(dᵢ² + dⱼ²) − 16·dᵢdⱼ/(mᵢ+mⱼ) + 4 ) ]

**STATEMENT CORRECTION (verified against DHS arXiv:2606.14550 LaTeX source, Table 2, lines
176–177):** the leading "2 +" is OUTSIDE the square root; an earlier version of this file had the
whole expression under the radical. Source LaTeX: `2+\sqrt{2((d_i-1)^2+(d_j-1)^2+m_i m_j-d_i d_j)}`
and `2+\sqrt{2(d_i^2+d_j^2)-16d_i d_j/(m_i+m_j)+4}`.

**IMPORTANT — verify exact formulas** against the original sources before any run: Brankov–Hansen–
Stevanović, "Automated conjectures on upper bounds for the largest Laplacian eigenvalue of graphs",
Linear Algebra Appl. 414 (2006); and Damnjanović–Ha–Stevanović, arXiv:2606.14550 (June 2026), whose
Theorem 1.1 states these are the ONLY two of the 68 auto-generated bounds still open (22 proved,
44 refuted across three campaigns: Al-Yakoob–Ghebleh–Kanso–Stevanović 2024; Taieb et al. LION 2025;
DHS 2026).

**Witness.** One connected graph violating the bound. Verifier: build L = D − A, compute μ exactly
(rational characteristic polynomial + Sturm isolation for the certificate; float for search), compute
RHS per edge. Counterexamples in DHS 2026 came from **equitable-partition quotient graphs** — search
that structured family (MC search under-explores it), plus ILP/annealing over degree/mᵢ profiles.

**Proof direction.** DHS proved 22 bounds via Collatz–Wielandt-style arguments; attempt the same for
44/46 if search convinces you they are true.
